🚀 Astro Framework – Getting Started

This repository contains a basic starter project built using the Astro Framework. It serves as a foundation for learning and experimenting with modern frontend development using Astro.

📌 Project Overview

Astro is a modern static site builder designed for fast performance and flexible UI development. It allows developers to build websites using multiple frontend frameworks while shipping minimal JavaScript by default.

This project was created as an initial hands-on implementation to understand:

Project structure in Astro
Component-based architecture
Development workflow
Build & deployment process
⚙️ Prerequisites

Before getting started, ensure you have the following installed:

Node.js (v18 or later recommended)
npm (comes with Node.js) or yarn
Git

You can verify installation using:

node -v
npm -v
git --version
📦 Installation & Setup

Clone the repository:

git clone https://github.com/EbrahimHeggy/Astro-Framework.git

Navigate into the project directory:

cd Astro-Framework

Install dependencies:

npm install
🚀 Running the Project

Start the development server:

npm run dev

Then open your browser and visit:

http://localhost:4321
🏗️ Build for Production

To generate an optimized production build:

npm run build

Preview the production build locally:

npm run preview
